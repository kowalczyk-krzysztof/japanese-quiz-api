# Japanese Quiz

Generate a random 2-kanji word and check whether it exists

### Output:

```typescript
{
 word: string,
 wordExists: boolean,
 reading: string,
 definitions: Array<string>
}
```

### Example output:

```json
{
  "word": "渉外",
  "wordExists": true,
  "reading": "しょうがい",
  "definitions": ["public relations", "client liaison", "client relations"]
}
```
